#  Smart Study Planner AI Assistant (Java)

An intelligent Java-based chatbot that helps students create optimized daily study schedules with motivational AI quotes and performance tracking.

##  Features
-  Multi-user support (separate plan files per user)
-  Chatbot-style interaction ("create plan for 6 hours")
-  Smart subject time allocation based on difficulty
-  Performance tracking for each subject
-  AI-style motivational quote generator (simulated GenAI)
-  File storage for each user's study plan and progress

##  Technologies Used
- Core Java  
- OOP Principles  
- Collections Framework  
- File Handling  
- Pattern Matching (Chatbot logic)

##  How to Run
```bash
javac SmartStudyPlannerAI_Assistant.java
java SmartStudyPlannerAI_Assistant
